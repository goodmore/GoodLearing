# GoodLearning 机器学习成长之路
> 个人机器学习学习路径仓库，全程记录从零入门 → 神经网络 → 强化学习 → 深度学习实战 的完整成长轨迹，沉淀代码、笔记、踩坑总结与算法复现。

## 📚 项目介绍
本仓库用于**系统性梳理机器学习学习路线**，同时作为个人成长存档：
- 记录机器学习/深度学习/强化学习 完整学习笔记
- 从零手写复现经典算法：线性回归、神经网络、Q-learning、SARSA、A3C 等
- 留存每阶段代码练习、实验demo、易错点复盘
- 持续更新学习心得、知识点总结、维度踩坑记录
- 适合自己复盘回顾，也可作为新手入门参考路径

## 🗂️ 仓库内容结构
```
GoodLearning/
├── 机器学习基础/        # 线性回归、逻辑回归、梯度下降、正则化等
├── 深度学习入门/        # 手写神经网络、权重初始化、前向/反向传播
├── 强化学习实战/        # GridWorld 网格环境、Q-learning、SARSA、A3C 算法复现
├── 知识点笔记/          # 维度讲解、矩阵形状、广播机制、激活函数等学习总结
├── 练习Demo/            # 日常刷题、小实验、调参测试代码
└── README.md           # 项目说明 & 学习进度归档
```
